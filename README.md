# College Companion

A modern, Android-first college management application built with Flutter.

## Features

* 📚 Attendance Management
* 🗓 Timetable
* 📅 Academic Calendar
* 📝 Assignment Tracker
* 📊 Internal Marks
* 📈 Safe Bunk Calculator
* ☁ Offline First with Automatic Cloud Sync
* 🔒 Secure Google Authentication
* 🌙 Dark Theme Only

---

## Tech Stack

### Frontend

* Flutter
* Dart
* Material Design 3

### State Management

* Riverpod

### Local Database

* Drift (SQLite)

### Backend

* Supabase
* PostgreSQL

### Authentication

* Google Sign-In

---

## Architecture

* Feature First
* Repository Pattern
* Offline First
* Local First
* Modular Design
