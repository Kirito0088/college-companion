# Material Design 3 Reference

## Purpose

Summary of Material Design 3 principles used by College Companion.

---

# Theme

Dark Theme only.

---

# Design Principles

- Expressive
- Accessible
- Consistent
- Adaptive

---

# Components

Use Material components whenever possible.

Examples:

- NavigationBar
- FilledButton
- Card
- Snackbar
- BottomSheet
- Dialog
- TextField

---

# Motion

Motion should communicate state.

Avoid decorative animations.

---

# Typography

Use a single typography system.

Maintain visual hierarchy.

---

# Elevation

Use subtle elevation.

Avoid heavy shadows.

---

# Icons

Material Symbols Rounded only.

---

# Accessibility

Maintain proper contrast.

Large touch targets.

Readable typography.

---

# Goal

The app should feel like a polished Material Design 3 application.
